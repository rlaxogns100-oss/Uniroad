# 학교생활기록부 심층 분석 보고서 — 작성 매뉴얼

> **참고 보고서**: `frontend/src/pages/SchoolRecordOneTimeReportPage.tsx`
>
> 이 매뉴얼은 위 파일에 구현된 보고서를 **기준 템플릿**으로 삼습니다.
> 새로운 학생의 보고서를 생성할 때, 해당 코드의 레이아웃·디자인·컴포넌트 구조를 **그대로 유지**하면서 학생 데이터만 교체하여 동일한 포맷의 4페이지 리포트를 산출합니다.
> 디자인 톤(색상·폰트·여백·카드 스타일 등)은 참고 보고서와 **완전히 동일**하게 유지하십시오.

### 1-2. 교과 성적 흐름 라인 차트 (좌측)

> 학생의 학기별 교과 등급 추이를 보여주는 꺾은선 그래프

#### Y축 (등급 축)

| 항목 | 규칙 |
|------|------|
| 최상단 라벨 | 학생의 **최고 등급** (예: 1등급이면 `1`) |
| 최하단 라벨 | 학생의 **최저 등급** (예: 4등급이면 `4`) |
| 중간 라벨 | 최고~최저 사이의 등급을 **균등 간격**으로 배치 |
| 가로 점선 | 각 등급 라벨 위치에 회색 점선 |

**예시**: 최고 1등급, 최저 4등급 → 가로줄 4개 (1, 2, 3, 4 균등 배치)
**예시**: 최고 2등급, 최저 5등급 → 가로줄 4개 (2, 3, 4, 5 균등 배치)
**예시**: 최고 1등급, 최저 6등급 → 가로줄 6개 (1, 2, 3, 4, 5, 6 균등 배치)

#### X축 (학기 축)

| 항목 | 규칙 |
|------|------|
| 첫 번째 세로줄 | 학생의 **가장 첫 학기** (현재 코드 기준 `1-1` 위치에 배치) |
| 마지막 세로줄 | 학생의 **가장 마지막 학기** (현재 코드 기준 `3-2` 위치에 배치) |
| 중간 세로줄 | 첫~마지막 사이 학기들을 **균등 간격**으로 배치 |
| 세로 점선 | 각 학기 위치에 회색 점선 |

**예시**: 1-1 ~ 3-2 (6학기) → 세로줄 6개 (1-1, 1-2, 2-1, 2-2, 3-1, 3-2)
**예시**: 1-1 ~ 2-2 (4학기) → 세로줄 4개 (1-1, 1-2, 2-1, 2-2), 첫 줄은 현 1-1 위치, 마지막 줄은 현 3-2 위치에, 나머지 균등 배치

#### 데이터 라인

| 요소 | 규칙 |
|------|------|
| 개별 과목 라인 | 국어·수학·영어·사회·과학 각각 얇은 선(opacity 0.25)으로 표시. 해당 학기에 성적이 없는 과목은 선 생략 |
| 전교과 평균 라인 | 굵은 진한 남색(#2E5C8A) 실선. 해당 학기 모든 과목 등급의 평균 |
| 국영수사과 평균 라인 | 굵은 하늘색(#A8D0E0) 실선. 해당 학기 주요 5과목 등급의 평균 |
| 데이터 포인트 | 각 라인의 학기별 지점에 원형 점(●) 표시 |
| 범례 | 그래프 우상단 내부에 `━ 전교과`, `━ 국영수사과` 표시 |

### 1-3. 6대 핵심 지표 레이더 차트 (우측)

> 학생의 종합 역량을 6각형 레이더 차트로 시각화

#### 6개 축 (12시 방향부터 시계방향)

| 순서 | 축 이름 | 평가 기준 |
|------|---------|----------|
| 1 (12시) | 학업역량 | 교과 등급 안정성 + 상승 추세 |
| 2 (2시) | 탐구 깊이 | 세특 서술의 분석·실증 수준 |
| 3 (4시) | 진로 연결성 | 희망 전공과 교과·비교과 활동의 일관성 |
| 4 (6시) | 공동체역량 | 자율활동·동아리에서의 협업·리더십 기록 |
| 5 (8시) | 창의융합역량 | 교과 간 개념 연결, 융합적 사고 |
| 6 (10시) | 자기주도성 | 자발적 탐구 시작·확장 여부 |

#### 점수 산출

| 항목 | 규칙 |
|------|------|
| 각 축 값 | 0~1 범위 (1이 만점, 육각형 꼭짓점에 해당) |
| 중앙 총점 | 아래 **가중 합산 공식**으로 산출 → 100점 만점으로 환산하여 `XX점` 표시 |
| 육각형 배경 | 4단계 동심 육각형 (100%, 75%, 50%, 25%) |
| 데이터 영역 | 파란색(#5B9BD5) 반투명 다각형 + 꼭짓점에 원형 점 |

#### 가중 합산 공식

> **학업역량(내신 성적)에 가중치 3배**를 부여한다.

```
총점 = (학업역량 × 3 + 탐구깊이 × 1 + 진로연결성 × 1 + 공동체역량 × 1 + 창의융합역량 × 1 + 자기주도성 × 1) / 8 × 100
```

| 축 | 가중치 |
|----|--------|
| 학업역량 | **3** |
| 탐구 깊이 | 1 |
| 진로 연결성 | 1 |
| 공동체역량 | 1 |
| 창의융합역량 | 1 |
| 자기주도성 | 1 |

## 2페이지: Part 02 진단과 보완 — 핵심 강점

### 개요

| 항목 | 규칙 |
|------|------|
| 뱃지 | `Part 02 진단과 보완` |
| 섹션 제목 | `학교생활기록부 핵심 강점` |
| **항목 수** | **3~4개** (학생의 생기부를 분석하여 가장 돋보이는 강점 3~4가지를 자율 선정) |
| 번호 뱃지 색상 | 파란색(#5B9BD5) |

> **강점 항목의 주제는 고정이 아니다.**
> 학생마다 강점이 다르므로, 생기부를 분석하여 해당 학생에게 가장 적합한 강점 주제를 자율적으로 선정한다.
> (예: 학업 역량, 공동체 역량, 심화탐구흐름, 리더십, 교과 융합력, 자기주도 학습, 진로 일관성 등)

### 각 강점 항목의 공통 구조

| 영역 | 작성 규칙 |
|------|----------|
| 번호 뱃지 | 파란색(#5B9BD5) 원에 순번 |
| 제목 | 강점 키워드 (예: `학업 역량`, `심화탐구흐름` 등) |
| 부제 | `— [출처 영역]` (예: `교과 성적 전반`, `2학년 자율활동`, `세특 전반` 등) |
| 설명 텍스트 | 1~2문장. 해당 강점이 왜 돋보이는지 요약 |
| 시각화/본문 | 아래 **시각화 방법** 중 적절한 것을 선택하여 배치 |

### 시각화 방법 (자율 선택)

> 각 강점 항목에 가장 적합한 시각화 방법을 자율적으로 판단하여 사용한다.
> 단, 아래 **필수 조건**을 반드시 충족해야 한다.

#### 필수 조건

| 조건 | 규칙 |
|------|------|
| 플로우차트 | 강점 영역 전체에서 **최소 1개** 포함 |
| 좌측 세특 인용 / 우측 분석 형식 | 강점 영역 전체에서 **최소 2개** 항목에 사용 |

#### 사용 가능한 시각화 유형

| 유형 | 설명 | 적합한 경우 (예시) |
|------|------|-------------------|
| **좌측 세특 / 우측 분석** | 좌측에 생기부 원문 인용(이탤릭, 따옴표), 우측에 2~3문장 분석 | 특정 세특 구절의 강점을 부각할 때 (공동체 역량, 리더십, 교과 심화 등) |
| **히트맵 테이블** | 과목×학기 격자에 등급 표시, 등급별 배경색 차등 | 학업 역량·성적 안정성을 보여줄 때 |
| **플로우차트** | 좌→우 흐름도. 노드(시작/분기/활동/합류/최종)와 화살표로 구성 | 탐구 흐름·활동 연결 구조를 보여줄 때 |
| **가로 바 차트** | 항목별 수평 막대 비교 | 비율·점수를 학년별/영역별로 비교할 때 |
| **비교 표** | 2열 비교 (before/after, 학기별 등) | 시기별 변화·성장을 대비할 때 |

#### 좌측 세특 / 우측 분석 형식 상세

| 영역 | 규칙 |
|------|------|
| 좌측 | 생기부 원문 인용 — 이탤릭체, 따옴표(`""`)로 감싸기. 해당 강점이 가장 잘 드러나는 구절 선택 |
| 우측 | 2~3문장 분석. 왜 이 부분이 강점인지, 전공과의 연결성, 차별화 포인트 설명 |

#### 플로우차트 작성 규칙

| 요소 | 형태 | 색상 | 의미 |
|------|------|------|------|
| 시작 노드 | 둥근 직사각형 | 진한 남색(#2E5C8A) 배경, 흰 텍스트 | 탐구의 출발점 |
| 분기 노드 | 마름모(◇) | 빨간(#C0392B) 테두리 | 진로 탐색/전공 심화 등 분기점 |
| 활동 노드 (동아리) | 둥근모서리 직사각형 | 연파랑(#EBF4FA) 배경, 파랑(#5B9BD5) 테두리 | 동아리/비교과 활동 |
| 활동 노드 (교과) | 둥근모서리 직사각형 | 연파랑(#EBF4FA) 배경, 하늘색(#7FB3D5) 테두리 | 교과 세특 활동 |
| 합류 노드 | 둥근 직사각형 | 파란(#5B9BD5) 배경, 흰 텍스트 | 여러 활동이 합쳐지는 핵심 성과 |
| 최종 성과 | 둥근 직사각형 | 진한 남색(#2E5C8A) 배경, 흰 텍스트 | 최종 탐구 결과물 |
| 화살표 | 회색(#94A3B8) 실선 | 위에 맥락 텍스트(파란색) | 활동 간 연결 이유 |
| 좌→우 흐름 | 시작 → 분기 → 학년별 활동 → 합류 → 최종 성과 | |
| 상하 분기 | 위쪽 = 동아리/비교과, 아래쪽 = 교과 세특 | |
| 학년 구분 | 세로 점선으로 학년 경계 표시, 상단에 학년 라벨 | |

## 3페이지: Part 02 진단과 보완 — 핵심 약점

### 개요

| 항목 | 규칙 |
|------|------|
| 뱃지 | `Part 02 진단과 보완` (2페이지와 동일 파트) |
| 섹션 제목 | `학교생활기록부 핵심 약점` |
| **항목 수** | **3~4개** (학생의 생기부를 분석하여 가장 보완이 필요한 약점 3~4가지를 자율 선정) |
| 번호 뱃지 색상 | 빨간색(#C0392B) |

> **약점 항목의 주제는 고정이 아니다.**
> 학생마다 약점이 다르므로, 생기부를 분석하여 해당 학생에게 가장 보완이 필요한 약점 주제를 자율적으로 선정한다.
> (예: 세특 서술 깊이, 전공 연결성, 성장성, 탐구 방법론, 교과 편중, 비교과 부족, 후속 탐구 부재 등)

### 각 약점 항목의 공통 구조

| 영역 | 작성 규칙 |
|------|----------|
| 번호 뱃지 | 빨간색(#C0392B) 원에 순번 |
| 제목 | 약점 키워드 (예: `세특 서술 깊이 변화`, `전공 연결성` 등) |
| 부제 | `— [출처 영역]` (예: `세특 전반`, `1학년 영어 세특`, `진로활동 1~3학년` 등) |
| 설명 텍스트 | 1~2문장. 해당 약점이 왜 문제인지, 어떤 점이 부족한지 요약 |
| 시각화/본문 | 아래 **시각화 방법** 중 적절한 것을 선택하여 배치 |

### 시각화 방법 (자율 선택)

> 각 약점 항목에 가장 적합한 시각화 방법을 자율적으로 판단하여 사용한다.
> 단, 아래 **필수 조건**을 반드시 충족해야 한다.

#### 필수 조건

| 조건 | 규칙 |
|------|------|
| 플로우차트 | 약점 영역 전체에서 **최소 1개** 포함 |
| 좌측 세특 인용 / 우측 분석 형식 | 약점 영역 전체에서 **최소 2개** 항목에 사용 |

#### 사용 가능한 시각화 유형

| 유형 | 설명 | 적합한 경우 (예시) |
|------|------|-------------------|
| **좌측 세특 / 우측 피드백** | 좌측에 생기부 원문 인용(이탤릭, 따옴표), 우측에 2~3문장 개선 피드백 | 특정 세특 구절의 약점을 지적할 때 (전공 연결성, 서술 깊이 등) |
| **약점 플로우차트** | 강점 플로우차트와 동일 구조 + 빨간 점선(연결 부재 구간) 추가 | 진로 전환·탐구 흐름의 단절을 보여줄 때 |
| **가로 누적 바 차트** | 항목별 비율 비교 (예: 감상형 vs 실증형) | 학년별 서술 수준 변화를 비교할 때 |
| **비교 표** | 2열 비교 (현재 vs 개선안, 시기별 등) | 구체적 보완 방향을 제시할 때 |

#### 좌측 세특 / 우측 피드백 형식 상세

| 영역 | 규칙 |
|------|------|
| 좌측 | 생기부 원문 인용 — 이탤릭체, 따옴표(`""`)로 감싸기. 약점이 가장 잘 드러나는 구절 선택 |
| 우측 | 2~3문장 피드백. 왜 이 부분이 약점인지, 어떻게 보완할 수 있는지 구체적으로 제시 |

#### 약점 플로우차트 추가 규칙

> 기본 구조는 강점 플로우차트와 동일하나, **약점 표시**가 추가됨

| 요소 | 형태 | 색상 | 의미 |
|------|------|------|------|
| 정상 연결 화살표 | 회색 실선 | #94A3B8 | 자연스러운 연결 |
| 약점 연결 화살표 | 빨간 점선 | #C0392B, strokeDasharray | 연결이 부재하거나 약한 구간 |
| 약점 주석 텍스트 | 빨간 텍스트 | #C0392B | 예: `왜 전환?`, `(연결 부재)`, `후속 탐구?` |
| 결론 부재 노드 | 빨간 점선 둥근 직사각형 | #C0392B 점선 테두리 | 기록이 없거나 보완이 필요한 최종 단계 |
| 범례 | 하단에 `━ 자연스러운 연결` / `┅ 연결 부재 (보완 필요)` |

#### 가로 누적 바 차트 작성 규칙 (예: 감상형 vs 실증형)

| 항목 | 규칙 |
|------|------|
| 행 수 | 비교 대상 수 (예: 학년별 3행) |
| 좌측 라벨 | 비교 대상명 (예: `1학년`, `2학년`, `3학년`) |
| 바 구성 | 2가지 항목의 비율을 좌·우로 누적 표시 (합계 100%) |
| 범례 | 하단에 각 항목의 색상·명칭 표기 |
