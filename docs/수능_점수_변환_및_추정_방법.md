## 2. 기본 점수 변환 방법

### 2.1 등급 → 표준점수 및 백분위 변환

**적용 과목**: 국어, 수학, 탐구영역

등급만 알고 있을 때, 해당 등급의 **중간 백분위**를 기준으로 표준점수를 추정합니다.

#### 등급별 중간 백분위 기준표

| 등급 | 백분위 범위 | 중간 백분위 |
|------|-------------|-------------|
| 1등급 | 96~100% | 98% |
| 2등급 | 89~96% | 92% |
| 3등급 | 77~89% | 83% |
| 4등급 | 60~77% | 68% |
| 5등급 | 40~60% | 50% |
| 6등급 | 23~40% | 31% |
| 7등급 | 11~23% | 17% |
| 8등급 | 4~11% | 7% |
| 9등급 | 0~4% | 2% |

#### 계산 절차

1. 등급에 해당하는 중간 백분위 확인
2. 2026 수능 성적분포표에서 해당 백분위에 가장 가까운 표준점수 조회
3. 정확한 값이 없으면 가장 가까운 값 사용

**예시**: 국어 1등급
- 중간 백분위: 98%
- 2026 수능 국어 98% → 표준점수 138점
- 결과: **1등급 / 표준점수 138 / 백분위 98**

---

### 2.2 표준점수 → 등급 및 백분위 변환

**적용 과목**: 국어, 수학, 탐구영역

표준점수를 알고 있을 때, 2026 수능 성적분포표에서 직접 조회합니다.

#### 계산 절차

1. 2026 수능 성적분포표에서 해당 표준점수 검색
2. 정확한 값이 있으면 해당 백분위와 등급 반환
3. 정확한 값이 없으면 가장 가까운 표준점수의 정보 사용

**예시**: 국어 표준점수 140점
- 2026 수능 국어 표준점수 140점 → 백분위 99%, 1등급
- 결과: **표준점수 140 / 백분위 99 / 1등급**

---

### 2.3 백분위 → 표준점수 및 등급 변환

**적용 과목**: 국어, 수학, 탐구영역

백분위를 알고 있을 때, 2026 수능 성적분포표에서 조회합니다.

#### 계산 절차

1. 2026 수능 성적분포표에서 해당 백분위 검색
2. 정확한 값이 있으면 해당 표준점수와 등급 반환
3. 정확한 값이 없으면 가장 가까운 백분위의 정보 사용

**예시**: 수학 백분위 95%
- 2026 수능 수학 백분위 95% → 표준점수 132점, 2등급
- 결과: **백분위 95 / 표준점수 132 / 2등급**

### 2.4 원점수 → 표준점수, 백분위, 등급 변환

#### 2.4.1 국어 / 수학 (100점 만점)

국어와 수학은 **선택과목에 따라** 같은 원점수라도 표준점수가 다릅니다.

##### 선택과목 종류
- **국어**: 화법과작문, 언어와매체
- **수학**: 확률과통계, 미적분, 기하

##### 계산 방법

1. 2026 수능 국어/수학 등급컷 데이터에서 선택과목별 원점수 기준표 확인
2. 입력된 원점수가 등급컷에 정확히 일치하면 해당 등급의 표준점수, 백분위 반환
3. 등급컷 사이에 있으면 **선형 보간법**으로 추정

**선형 보간법이란?**  
두 등급컷 사이의 점수는 비례식을 이용해 중간값을 계산하는 방법입니다.

**예시**: 국어(화법과작문) 원점수 92점
- 만점(100점): 표준점수 142, 백분위 99%
- 1등급컷(90점): 표준점수 133, 백분위 96%
- 92점은 90~100점 사이 → 선형 보간
  - 표준점수: 133 + (142-133) × (92-90)/(100-90) = 135점
  - 백분위: 96 + (99-96) × (92-90)/(100-90) = 97%
  - 등급: 1등급 (90점 이상)
- 결과: **1등급 / 표준점수 135 / 백분위 97**

---

#### 2.4.3 탐구영역 (50점 만점)

탐구 과목은 **50점 만점**이며, 2026 수능 실제 채점 결과를 기반으로 변환합니다.

##### 지원 과목
- **사회탐구 (9과목)**: 경제, 동아시아사, 사회문화, 생활과윤리, 세계사, 세계지리, 정치와법, 한국지리, 윤리와사상
- **과학탐구 (8과목)**: 물리학Ⅰ, 물리학Ⅱ, 화학Ⅰ, 화학Ⅱ, 생명과학Ⅰ, 생명과학Ⅱ, 지구과학Ⅰ, 지구과학Ⅱ

##### 계산 방법

1. 원점수가 50점을 초과하면 100점 만점 기준으로 입력된 것으로 판단
   - 예: 90점 입력 → 90 × 50/100 = 45점으로 자동 변환
2. 2026 수능 탐구영역 원점수-표준점수-백분위 매핑 테이블에서 조회
3. 정확한 원점수가 있으면 해당 표준점수, 백분위, 등급 반환
4. 정확한 원점수가 없으면 가장 가까운 원점수의 데이터 사용

**예시 1**: 생활과윤리 원점수 90점 입력 (100점 만점으로 잘못 입력)
- 자동 변환: 90 × 50/100 = 45점
- 2026 수능 생활과윤리 45점 → 표준점수 66, 백분위 95%, 1등급
- 결과: **1등급 / 표준점수 66 / 백분위 95 / 원점수 45**

**예시 2**: 사회문화 원점수 43점 입력
- 2026 수능 사회문화 43점 → 표준점수 64, 백분위 94%, 2등급
- 결과: **2등급 / 표준점수 64 / 백분위 94 / 원점수 43**

**예시 3**: 생명과학Ⅰ 원점수 44점 입력 (매핑 테이블에 정확한 값 없음)
- 가장 가까운 원점수 조회: 43점(표준점수 68, 백분위 98%, 1등급) 또는 45점(표준점수 70, 백분위 99%, 1등급)
- 44점에 더 가까운 45점 데이터 사용
- 결과: **1등급 / 표준점수 70 / 백분위 99 / 원점수 44 (추정)**

## 3. 데이터가 부족한 경우 처리 방법

### 3.1 선택과목 정보 없을 때

**국어/수학 원점수 변환 시** 선택과목을 입력하지 않으면 다음 기본값을 사용합니다:

- **국어**: 화법과작문 (표준 선택과목)
- **수학**: 확률과통계 (표준 선택과목)

**권장사항**: 정확한 분석을 위해 선택과목을 함께 알려주세요.

---

### 3.2 정확한 표준점수/백분위가 없을 때

2026 수능 성적분포표에 정확한 값이 없을 경우:

1. **가장 가까운 값 사용**
   - 표준점수 141점 요청 → 140점 또는 142점 중 가까운 값
   - 백분위 97.5% 요청 → 97% 또는 98% 중 가까운 값

2. **국어/수학 원점수의 경우 선형 보간**
   - 등급컷 사이 원점수는 비례식으로 정확히 계산

---

### 3.3 탐구 과목 원점수 데이터 부족

일부 탐구 과목의 경우 모든 원점수에 대한 데이터가 없을 수 있습니다.

**처리 방법**:
- 입력한 원점수에 가장 가까운 원점수의 데이터 사용
- 추정된 경우 명시: "(추정)" 표시

**예시**: 물리학Ⅰ 원점수 44점 (데이터에 43점, 45점만 있는 경우)
- 45점 데이터 사용 (더 가까움)
- 결과: **2등급 / 표준점수 66 / 백분위 96 / 원점수 44 (추정)**

## 4. 대학별 환산 점수 계산

각 대학은 수능 점수를 자체 기준으로 환산합니다. 모든 계산은 **2026학년도 정시 모집요강**을 기준으로 합니다.

---

### 4.1 서울대학교 정시 환산 (1000점 만점)

#### 적용 기준

- **2026학년도 서울대학교 정시 일반전형 안내**
- 수능 100% 반영 (일반전형 기준)
- 실기/기타 있는 모집단위는 별도 배점

#### 일반전형 (인문/자연/공학/의약) 환산 공식

```
환산점수 = 국어표준점수 + (수학표준점수 × 1.2) + (탐구표준점수평균 × 0.8) + 과탐가산점 - 감점
```

##### 주요 계산 요소

**1. 탐구 2과목 평균**
- (탐구1 표준점수 + 탐구2 표준점수) ÷ 2

**2. 과학탐구 가산점 (자연/공학/의약계만)**
- 과학Ⅰ + 과학Ⅰ: 가산점 없음
- 과학Ⅰ + 과학Ⅱ: 3점 가산
- 과학Ⅱ + 과학Ⅱ: 5점 가산

**3. 감점 요소**

| 구분 | 감점 기준 |
|------|----------|
| 영어 1등급 | 0점 |
| 영어 2등급 | -0.5점 |
| 영어 3등급 | -1점 |
| 영어 4등급 | -2점 |
| 영어 5등급 | -4점 |
| 영어 6등급 | -6점 |
| 영어 7등급 | -8점 |
| 영어 8등급 | -10점 |
| 영어 9등급 | -12점 |

| 구분 | 감점 기준 |
|------|----------|
| 한국사 1~4등급 | 0점 |
| 한국사 5등급 | -0.2점 |
| 한국사 6등급 | -0.4점 |
| 한국사 7등급 | -0.6점 |
| 한국사 8등급 | -0.8점 |
| 한국사 9등급 | -1점 |

**제2외국어/한문 감점 (인문계만)**
- 제2외국어/한문 응시 안 함: -3점
- 응시했으면 감점 없음

#### 계산 예시 (자연계)

**입력 성적**:
- 국어(언어와매체) 1등급 → 표준점수 138
- 수학(미적분) 1등급 → 표준점수 133
- 생명과학Ⅰ 1등급 → 표준점수 70
- 지구과학Ⅰ 1등급 → 표준점수 66
- 영어 2등급
- 한국사 3등급

**계산 과정**:
1. 탐구 평균: (70 + 66) ÷ 2 = 68
2. 기본 점수: 138 + (133 × 1.2) + (68 × 0.8) = 138 + 159.6 + 54.4 = 352
3. 과탐 가산점: 과학Ⅰ+Ⅰ → 0점
4. 영어 감점: 2등급 → -0.5점
5. 한국사 감점: 3등급 → 0점
6. **최종 점수: 352 + 0 - 0.5 = 351.5점 / 1000점**

### 4.2 연세대학교 정시 환산 (1000점 만점)

#### 적용 기준

- **2026학년도 연세대학교 정시 수능 활용지표**
- 수능 100% 반영
- 탐구 2과목 변환표준점수 활용

#### 자연계 환산 공식

```
환산점수 = (국어표준점수 + (수학표준점수×1.5) + 영어점수 + (탐구변환표준×1.5)) / 분모 × 1000 × 1.03
```

##### 주요 계산 요소

**1. 탐구 변환표준점수**
- 연세대 자체 변환표 사용 (과학탐구 백분위 → 변환표준점수)
- 2과목 합산

**예시 변환표** (과학탐구, 일부 발췌):

| 백분위 | 변환표준점수 |
|--------|-------------|
| 100 | 68.00 |
| 99 | 67.75 |
| 98 | 67.50 |
| 97 | 67.25 |
| 96 | 67.00 |
| 95 | 66.75 |
| ... | ... |

**2. 영어 등급별 점수**

| 등급 | 점수 |
|------|------|
| 1등급 | 100 |
| 2등급 | 95 |
| 3등급 | 87.5 |
| 4등급 | 75 |
| 5등급 | 60 |
| 6등급 | 40 |
| 7등급 | 25 |
| 8등급 | 12.5 |
| 9등급 | 5 |

**3. 분모 (계열별 상이)**
- 인문: 800
- 자연: 900
- 의약: 900

**4. 과탐 가산점 3%**
- 자연/의약계: 최종 점수 × 1.03
- 인문계: 사탐 가산점 × 1.03

#### 계산 예시 (자연계)

**입력 성적**:
- 국어 표준점수 138
- 수학 표준점수 133
- 생명과학Ⅰ 백분위 99% → 변환표준점수 67.75
- 지구과학Ⅰ 백분위 96% → 변환표준점수 67.00
- 영어 2등급 → 95점

**계산 과정**:
1. 분자: 138 + (133 × 1.5) + 95 + ((67.75 + 67.00) × 1.5)
   = 138 + 199.5 + 95 + 202.125 = 634.625
2. 환산: 634.625 / 900 × 1000 = 705.14
3. 과탐 가산(3%): 705.14 × 1.03 = **726.29점 / 1000점**

### 4.3 고려대학교 정시 환산 (1000점 만점)

#### 적용 기준

- **2026학년도 고려대학교 정시 수능 반영방법**
- 수능 100% 반영
- 영어/한국사 감점제

#### 자연계 환산 공식

