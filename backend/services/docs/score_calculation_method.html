2026 수능 표준점수 및 백분위 산출 방식

버전: 1.0
최종 수정일: 2026년 1월 22일
작성: UniZ 입시 컨설팅 시스템

1. 개요

본 문서는 UniZ 시스템이 학생의 성적 정보를 처리하는 방식과 2026학년도 수능 데이터를 기반으로 한 표준점수, 백분위, 등급 산출 방법을 설명합니다.

2. 데이터 출처

  • 기준 연도: 2026학년도 대학수학능력시험
  • 데이터 범위: 국어, 수학, 영어, 한국사, 사회탐구(9과목), 과학탐구(8과목)
  • 데이터 유형:
    • 국어/수학: 표준점수-백분위-등급 매핑 테이블
    • 탐구 과목: 원점수-표준점수-백분위-등급 매핑 테이블
    • 국어/수학 선택과목: 원점수-표준점수-백분위-등급 등급컷 데이터

3. 성적 입력 형식

3.1 지원 형식

입력 방식 예시 설명
축약 형식 "나 11232야" 국어1-수학1-영어2-탐구1(3)-탐구2(2)
과목별 등급 "국어 1등급 수학 2등급" 과목명 + 등급
표준점수 "국어 표준점수 138" 과목명 + 표준점수
백분위 "수학 백분위 95" 과목명 + 백분위
원점수 "국어 화법과작문 92점" 과목명 + 선택과목 + 원점수

3.2 선택과목 처리

  • 국어: 화법과작문 / 언어와매체
  • 수학: 확률과통계 / 미적분 / 기하
  • 선택과목 미입력 시 기본값:
    • 국어: 화법과작문
    • 수학: 확률과통계

4. 산출 방식

4.1 등급 → 표준점수 / 백분위 변환

등급만 입력된 경우, 해당 등급의 중간 백분위를 사용하여 보수적으로 산출합니다.

등급별 백분위 범위 및 중간값

등급 백분위 범위 중간값 (사용) 설명
1등급 96~100% 98% 상위 4% 이내
2등급 89~96% 92% 상위 11% 이내
3등급 77~89% 83% 상위 23% 이내
4등급 60~77% 68% 상위 40% 이내
5등급 40~60% 50% 상위 60% 이내
6등급 23~40% 31% 상위 77% 이내
7등급 11~23% 17% 상위 89% 이내
8등급 4~11% 7% 상위 96% 이내
9등급 0~4% 2% 상위 100%

산출 로직:

  1. 입력된 등급의 중간 백분위 선택
  2. 해당 백분위에서 가장 가까운 표준점수 데이터 검색
  3. 검색된 표준점수-백분위-등급 반환

예시:

  • 입력: "국어 1등급"
  • 중간 백분위: 98%
  • 2026 수능 데이터 조회: 표준점수 138점, 백분위 98%, 등급 1
  • 출력: 1등급 / 표준점수 138 / 백분위 98

4.2 표준점수 → 백분위 / 등급 변환

표준점수가 입력된 경우, 2026 수능 데이터 테이블에서 직접 매칭합니다.

4.3 백분위 → 표준점수 / 등급 변환

백분위가 입력된 경우, 2026 수능 데이터 테이블에서 가장 가까운 값을 찾습니다.

4.4 원점수 → 표준점수 / 백분위 / 등급 변환 (국어/수학 선택과목)

국어와 수학의 선택과목은 원점수로 입력 가능하며, 선형 보간법을 사용하여 등급컷 사이의 점수를 추정합니다.

4.4.1 선형 보간법 (Linear Interpolation)

등급컷 데이터 사이의 점수는 다음 공식으로 계산합니다:

y = y1 + (y2 - y1) × (x - x1) / (x2 - x1)

여기서:
- x: 입력된 원점수
- x1, x2: 인접한 두 등급컷의 원점수
- y1, y2: 인접한 두 등급컷의 표준점수 또는 백분위
- y: 추정된 표준점수 또는 백분위

4.4.2 등급 결정 로직

원점수가 두 등급컷 사이에 위치하는 경우:

  • 더 낮은 등급 (하위 컷)을 부여
  • 예: 77점이 1등급컷(85점)과 2등급컷(73점) 사이 → 2등급 부여

4.4.3 예시: 수학 미적분 77점

데이터:

  • 1등급컷: 원점수 88점 → 표준점수 137점, 백분위 99%, 등급 1
  • 2등급컷: 원점수 73점 → 표준점수 123점, 백분위 92%, 등급 2

입력: 수학 미적분 77점

계산 과정:

1. 표준점수 계산:

표준점수 = 123 + (137 - 123) × (77 - 73) / (88 - 73)
         = 123 + 14 × 4 / 15
         = 123 + 3.73
         ≈ 127

2. 백분위 계산:

백분위 = 92 + (99 - 92) × (77 - 73) / (88 - 73)
       = 92 + 7 × 4 / 15
       = 92 + 1.87
       ≈ 94

3. 등급 결정:

  • 77점은 1등급컷(88점)보다 낮고 2등급컷(73점)보다 높음
  • 2등급 부여 (하위 컷 기준)

출력: 2등급 / 표준점수 127 / 백분위 94

4.5 영어 등급 처리 (절대평가)

영어는 절대평가 과목으로 표준점수가 존재하지 않습니다.

출력 형식:

영어: X등급 / 표준점수 없음(절대평가) / 백분위 Y

4.6 미입력 과목 추정

학생이 모든 과목을 입력하지 않은 경우, 입력된 과목들의 평균 백분위를 사용하여 미입력 과목을 추정합니다.

추정 로직:

  1. 입력된 과목들의 백분위 평균 계산
  2. 평균 백분위에 해당하는 표준점수/등급 검색
  3. 추정된 과목에 (추정) 표시

5. 데이터 정확성 및 제한사항

5.1 데이터 정확성

  • 본 시스템은 2026학년도 수능 공식 발표 데이터를 사용합니다.
  • 표준점수 테이블은 한국교육과정평가원 자료를 기준으로 합니다.

5.2 제한사항

  1. 원점수 변환의 추정치: 등급컷 사이의 원점수는 선형 보간을 사용하므로, 실제 점수와 약간의 오차가 있을 수 있습니다.
  2. 선택과목 미입력 시: 기본값(화법과작문, 확률과통계)이 자동 적용되므로, 실제 선택과목과 다를 수 있습니다.
  3. 미입력 과목 추정: 추정된 과목은 평균 백분위 기준이므로, 실제 성적과 차이가 있을 수 있습니다.
  4. 탐구 과목 특정되지 않은 경우: "탐구1", "탐구2"로만 입력된 경우, 사회문화 과목을 기준으로 추정합니다.

5.3 보수적 산출 원칙

  • 등급만 입력된 경우, 등급 범위의 중간값을 사용하여 과대평가를 방지합니다.
  • 원점수가 등급컷 사이에 있을 때, 더 낮은 등급을 부여하여 안전하게 추정합니다.

6. 사용 예시

예시 1: "나 11232야"

입력 해석:

  • 국어 1등급, 수학 1등급, 영어 2등급, 탐구1 3등급, 탐구2 2등급

산출 결과:

- 국어(화법과작문): 1등급 / 표준점수 138 / 백분위 98
- 수학(확률과통계): 1등급 / 표준점수 135 / 백분위 99
- 영어: 2등급 / 표준점수 없음(절대평가) / 백분위 82.5
- 탐구1: 3등급 / 표준점수 57 / 백분위 83
- 탐구2: 2등급 / 표준점수 58 / 백분위 92
