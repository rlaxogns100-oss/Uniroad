# 유니로드 로컬 실행 가이드

## 1. 환경 변수 설정

### 백엔드 (`backend/.env`)
다음 값을 **실제 발급받은 값**으로 수정하세요.

- **SUPABASE_URL**: [Supabase](https://supabase.com) 프로젝트 URL
- **SUPABASE_KEY**: Supabase anon (public) key
- **GEMINI_API_KEY**: [Google AI Studio](https://aistudio.google.com/)에서 발급
- **SUPABASE_JWT_SECRET**: Supabase 대시보드 → Settings → API → JWT Secret

### 프론트엔드 (`frontend/.env`)
로컬 개발 시 기본값 그대로 사용 가능 (백엔드가 `http://localhost:8000`일 때).

- `VITE_API_URL=http://localhost:8000`
- `VITE_API_BASE_URL=http://localhost:8000`

---

## 2. 서버 실행

### 백엔드 (터미널 1)
```bash
cd backend
source venv/bin/activate   # Windows: venv\Scripts\activate
python main.py
# 또는: uvicorn main:app --reload --host 0.0.0.0 --port 8000
```
- API 문서: http://localhost:8000/docs

### 프론트엔드 (터미널 2)
```bash
cd frontend
npm run dev
```
- 웹 앱: http://localhost:5173

---

## 3. 요약
| 구분    | 경로      | 실행 명령              |
|--------|-----------|-------------------------|
| 백엔드 | `backend/`| `source venv/bin/activate && python main.py` |
| 프론트 | `frontend/` | `npm run dev`        |

.env에 실제 Supabase/Gemini 키를 넣은 뒤 위 순서대로 실행하면 됩니다.
