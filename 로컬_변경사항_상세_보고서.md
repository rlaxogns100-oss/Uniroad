### 2.1 핵심 파일

#### `backend/services/multi_agent/score_system/excel_calculator.py`
**목적**: 엑셀 환산점수 공식을 100% 재현하기 위한 새 계산기

**주요 기능**:
- `load_data()`: formulas_final.json, deductions_final.json, conversion_tables_final.json 로드
- `get_conversion_table_key()`: 대학별 변환표준점수 테이블 키 반환
- `convert_to_standard()`: 백분위 → 변환표준점수 변환
- `calculate_score()`: 환산점수 계산 (표준점수/백분위/변환표준점수 타입 지원)

**데이터 구조**:
```python
# 입력 파라미터
formula_id: str,           # 공식 ID (1~282)
korean_std: float,         # 국어 표준점수
korean_pct: float,         # 국어 백분위
math_std: float,           # 수학 표준점수
math_pct: float,           # 수학 백분위
tamgu1_std: float,         # 탐구1 표준점수
tamgu1_pct: float,         # 탐구1 백분위
tamgu2_std: float,         # 탐구2 표준점수
tamgu2_pct: float,         # 탐구2 백분위
english_grade: int,        # 영어 등급 (1~9)
history_grade: int,        # 한국사 등급 (1~9)
is_science: bool,          # 과탐 여부
```

**반영방법 타입**:
- 1 = 표준점수
- 2 = 백분위
- 4 = 변환표준점수
- 9 = 등급
- 12 = 표준점수변환

```python

english_score = float(deduction.get("englishDeduction", 0))

    history_score = float(history_deductions[history - 1])

if english_table and 1 <= english <= 9:
    english_score = float(english_table[english - 1])

history_table = deduction.get("history", [])
if history_table and 1 <= history <= 9:
    history_score = float(history_table[history - 1])
```

#### 변경 3: "하향" 판정 추가
```python
# 기존
if safe and my_score >= safe:
    return "안정"

# 변경
if safe and my_score >= safe:
    excess_ratio = (my_score - safe) / safe if safe > 0 else 0
    if excess_ratio >= 0.01:  # 1% 이상 초과
        return "하향"
    return "안정"
```

#### 변경 5: 하향 기본 제외
